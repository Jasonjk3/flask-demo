# flask-demo
**flask框架通用后台架构**

该项目主要由flask+redis+mongodb构成
集成了flask众多组件:
Flask-APScheduler
Flask-Cors
Flask-HTTPAuth
Flask-Login
flask-mongoengine
Flask-PyMongo
Flask-Script
Flask-SQLAlchemy
Flask-Uploads
Flask-WTF

### 两种设计风格可供选择:
WEB 通用 经典MVC设计模式
RESTful API设计风格
后续将不断完善该项目
